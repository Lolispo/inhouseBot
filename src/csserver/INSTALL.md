# Install Guide to Get5 on Dathost

* Create dathost csgo server server
* Go to: Mods & Plugins
* Enable source mod on server
* Enable mysql 
* Download get5 sourcemod plugin https://github.com/splewis/get5/releases/tag/0.7.1
* Upload to dathost as zipfile

* (Restart)
